What this extension does:

  Whenever a web page is loaded, all of the images on it are replaced with a random image taken from LOLcats.

Why:

  Demonstrates how to run code on page load;  how to connect to a URL and load an XML file using an XMLHttpRequest; how to parse that XML file and get data from it; how to find DOM nodes in a content page; and how to alter those DOM nodes to change the contents of the page that is displayed.  Phew!  That's a lot!
